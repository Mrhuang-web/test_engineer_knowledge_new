# 02数据来源

```
1、数据均来自于es（通过 zz_data_sync_info配置的批次号，到es中匹配对应月份下对应批次然后回显）
2、目前除了空间_数据中心是入库到数据库，其他都是走es（后端读取，会对边省市区编码进行转换）  -- 现在转mpp

测试环境中操作：
	修改批次号，查询时会实时同步对应批次数据
	即修改batch_num字段前面的yyyymmdd即可
```
