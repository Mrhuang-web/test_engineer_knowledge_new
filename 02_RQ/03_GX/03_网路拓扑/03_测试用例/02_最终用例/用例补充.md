# 补充维度

```
1、上游不存在时，可否创建
2、创建是上游不存在，后面再补充上游 - 这时上下级可以建立联系吗
3、自动绘画时
	跨市   --> 省维度 - 呈现不同市之间的关联（即存在一个市主节点--其实就是市下区的站点，只不过跨市了）
	跨区   --> 市维度 - 呈现不同区之间的关联（即存在一个区主节点--其实就是区的站点，只不过跨区了）
	跨站点  --> 区维度 - 呈现不同站点之间的关联（即存在一个站点主节点）
	站点  --> 可以实现组件拖拽（仅展示站点信息）
	机房  --> 可以实现组件拖拽（仅展示机房信息）
4、组件的保存只是暂存，切换就会小时
5、左上角提交才算真正完成topo
```
