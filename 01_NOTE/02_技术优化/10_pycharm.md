# pycharm项目选择打开窗口方式设置

```
打开PyCharm。
进入设置：
    在Windows/Linux上，点击菜单栏的 File -> Settings。
    在macOS上，点击菜单栏的 PyCharm -> Preferences。
在设置窗口中，依次导航到 Appearance & Behavior -> System Settings。
在右侧找到 Project Opening 部分，您会看到一个名为 Open project in 的选项。
点击下拉菜单，您会有几个选择：
    New window：总是在新窗口中打开项目。
    Current window：总是在当前窗口中打开项目（替换当前项目）。
    Ask：每次都会询问您，显示选择对话框。
您只需要选择 Ask 选项，然后点击 OK 或 Apply 按钮保存设置即可。
```
