### 人脸门禁一体机接口URL汇总（按接口类别分类）
以下是文档中所有接口的请求URL及对应接口名称，按设备管理、人员管理、照片管理、识别记录四大类整理，均基于根地址 `http://设备IP:8090/` 拼接：

## 一、设备管理类接口
| 接口名称                | 请求Method | 请求URL                              |
|-------------------------|------------|-------------------------------------|
| 设置设备密码            | POST       | http://设备IP:8090/setPassWord       |
| 设备信息查询            | GET        | http://设备IP:8090/device/information |
| 设置设备时间            | POST       | http://设备IP:8090/setTime          |
| 语言切换                | POST       | http://设备IP:8090/device/setLanguage |
| 设置时区                | POST       | http://设备IP:8090/device/setTimeZone |
| 设备重启                | POST       | http://设备IP:8090/restartDevice    |
| 识别回调                | POST       | http://设备IP:8090/setIdentifyCallBack |
| 注册照片回调            | POST       | http://设备IP:8090/setImgRegCallBack |
| 远程控制输出            | POST       | http://设备IP:8090/device/openDoorControl |
| 信号输入设置            | POST       | http://设备IP:8090/device/setSignalInput |
| 会议与关门告警设置      | POST       | http://设备IP:8090/meetAndWarnSet   |
| 卡片设置                | POST       | http://设备IP:8090/cardInfoSet      |
| 事件回调                | POST       | http://设备IP:8090/device/eventCallBack |
| 获取门磁状态            | GET        | http://设备IP:8090/getDoorSensor    |

## 三、照片管理类接口
| 接口名称                | 请求Method | 请求URL                              |
|-------------------------|------------|-------------------------------------|
| 照片注册                | POST       | http://设备IP:8090/face/create      |
| 照片删除                | POST       | http://设备IP:8090/face/delete      |
| 照片更新                | POST       | http://设备IP:8090/face/update      |
| 照片查询                | POST       | http://设备IP:8090/face/find        |
| 拍照注册                | POST       | http://设备IP:8090/face/takeImg     |
| 清空人员注册照片        | POST       | http://设备IP:8090/face/deletePerson |

## 四、识别记录类接口
| 接口名称                | 请求Method | 请求URL                              |
|-------------------------|------------|-------------------------------------|
| 识别记录查询            | GET        | http://设备IP:8090/newFindRecords    |
| 识别记录删除            | POST       | http://设备IP:8090/newDeleteRecords |

### 说明
1. 所有接口均需传入设备密码（pass参数），初次使用需通过「设置设备密码」接口初始化密码（默认12345678）。
2. POST请求参数需放在body中，格式为 `application/x-www-form-urlencoded`；GET请求参数需拼接在URL后。
3. 设备IP需替换为实际部署的设备局域网IP，端口固定为8090。
