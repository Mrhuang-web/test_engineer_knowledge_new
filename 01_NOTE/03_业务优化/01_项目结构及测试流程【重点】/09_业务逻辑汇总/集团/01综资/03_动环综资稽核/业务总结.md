# 01功能说明

```
统计动环与综资匹配情况

大致逻辑：
	1、站点、机房、设备（仅数据中心-楼栋、通信枢纽楼）
	2、设备（仅动环设备下-11类）
		（变压器、发电机、ups设备、开关电源）
		（高压配电、低压交流配电、交流母线配电、低压直流配电、高压直流开关电源、高压直流配电、一体化能源柜）
	3、未匹配情况跳转至分析-统计查询中
		稽核汇总
		稽核明细（可跳转）
	4、稽核明细匹配
		机房：仅13类机房
		设备：仅11类设备
```

# 02数据来源

```
1、数据均来自于es（通过 zz_data_sync_info配置的批次号，到es中匹配对应月份下对应批次然后回显）
2、目前除了空间_数据中心是取动环数据库，其他都是走es
3、映射表、zz_data_sync_info（需要先修正好批次号）


涉及表：
    前置综资动环匹配入库表
        zz_to_rm_rm_device
        zz_to_rm_rm_area_room
        zz_to_rm_rm_area_dc
        zz_to_rm_rm_area_site
    最终业务展示表
        zz_audit_summary_device_type
        zz_audit_summary_site
        zz_audit_summary_room_type


需要有前置条件（参考分析-查询统计中）：
	1、即动环和综资要先有匹配关系 - 并且触发入库
	2、根据入库信息来统计匹配情况（否则匹配统计计算不到） - 匹配后会入库到最终表用于回显页面
```
