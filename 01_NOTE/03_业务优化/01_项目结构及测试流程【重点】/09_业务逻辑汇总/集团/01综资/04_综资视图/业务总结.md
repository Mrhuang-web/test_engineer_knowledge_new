# 01功能说明

```
统计综资原始数据中的站点属性数据与空间表的关联（仅数据中心、传输节点、通信机楼）
	用于汇聚机房转供电 - 跳转查看（查看站点的动环站点类型和入网日期）
```

# 02数据来源

```
1、数据均来自于es（通过 zz_data_sync_info配置的批次号，到es中匹配对应月份下对应批次然后回显）
2、主要涉及空间站点和站点属性两个索引，以及zz_to_rm_site_property表
3、映射表、zz_data_sync_info（需要先修正好批次号）


涉及表：
    站点和站点属性索引
    zz_to_rm_site_property表


业务逻辑：
	1、动环站点类型（取值站点属性索引），入网日期（取值空间站点索引）
	2、通过站点属性表为主 -> 然后再匹配空间站点内的数据 -> 组成一条数据（站点属性-zh_label -> 站点空间-int_id）入库到zz_to_rm_site_property
		目前已以res_code为主键入库（在es中站点属性的唯一标识，防止如入库被去重或过滤）


涉及服务：
	1、report，composite


触发curl：
	curl --location --request POST 'http://10.1.6.32:32218/v1/report/zzData/site/curlSitePropertyTask' \
--header 'User-Agent: Apifox/1.0.0 (https://apifox.com)' \
--header 'Content-Type: application/json' \
--data-raw '{}'
```
