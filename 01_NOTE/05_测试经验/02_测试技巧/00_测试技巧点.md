# 01现网或是测试环境触发无数据

```
以上海为例
	机柜断电分析，触发采集不到最新值
	

分析解决
	直接通过造数据库数据，来验证一些关联功能（具体还是要看开发的逻辑）
		上海机柜断电分析这块，主要是es拿数据，然后判断后，存放到mysql表
		然后前端定时请求，请求接口主要就是拿mysql实时表给到前端，前端进行判断
			因此可以直接插入数据库进行校验
	但是不足点
		不能验证到es是否真的获取到，是否能够更新到实时表，能不能落到数据库
```
